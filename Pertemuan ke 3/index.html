Pemformatan Teks

Teks Tebal menggunakan tag <b> atau <strong>.

Teks Miring menggunakan tag <i> atau <em>.

Teks Bergaris Bawah menggunakan tag <u> atau <ins>.

Teks Tercoret menggunakan tag <s> atau <del>.

Pangkat Atas: X2 menggunakan <sup>.

Pangkat Bawah: H2O menggunakan <sub>.

Marker Teks menggunakan tag <mark>.
